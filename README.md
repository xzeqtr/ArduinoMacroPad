# Arduino MacroPad

Простой контроллер (автокликер) на базе **Arduino Pro Micro** (т.к. умеет эмулировать HID-устройства (клавиатура/мышь)) с тремя вариантами прошивки — от базового до продвинутого с дисплеем и настройками.

---

## Варианты прошивки

В вариантах с кнопками в первую очередь надо обратить внимание на номера пинов, к которым подключены кнопки.

### ▶️ `simplest` — Простейший
- При подключении к ПК отправляет **одну заранее заданную клавишу**.
- Идеально для старта: минимум кода, максимум понятности.

### 🔘 `simple_button` — С кнопкой вкл/выкл
- Добавлена **физическая кнопка** для включения/отключения отправки нажатия.
- Удобно, когда нужно временно отключить макрос без переподключения устройства.
- Логика: нажал кнопку → макрос активен, нажал еще раз → пауза.

### 🖥️ `KeybMouseDisplay` — Продвинутый (клавиатура + мышь + дисплей)
- Поддержка **клавиатуры** и **клика мышью**.
- **Регулируемые задержки** для клавиатуры и мыши (своя кнопка для каждого устройства).
- **OLED-дисплей** (0,91", I2C) отображает:
  - Текущий статус (активен/пауза)
  - Значения задержек
---
## 🔧 Аппаратная часть
- Для варианта `simplest` достаточно **Arduino Pro Micro**
- Для варианта `simple_button` достаточно **Arduino Pro Micro**, **breadboard** и одной **кнопки**
- Вариант `KeybMouseDisplay` требует **Arduino Pro Micro**, две **breadboard**, четыре **кнопки**, 0,91" I2C дисплей и провода для подключения

В папке Device находится информация по сборке:
- Fritzing/ # Схема подключения в формате .fzz (Fritzing)
- Photos/ # Фотографии реальной сборки для наглядности
